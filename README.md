# aJ

Two‑user passwordless chat (J ↔ a) built for Deno Deploy.

## Run locally

```bash
deno task start
```

Then open http://localhost:8000 in two different browsers or devices. Choose user `J` in one and `a` in the other to chat.

## Deploy

Use this repo directly with Deno Deploy (entrypoint: `main.ts`).
